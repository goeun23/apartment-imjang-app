1. **프로젝트 생성**: `apartment-imjang-app` 디렉토리에 Next.js 프로젝트 초기화
2. **shadcn/ui 사용**: 빠른 개발을 위해 포함
3. **Mock 데이터 우선**: API 키 없이도 개발 가능하도록 Mock 데이터로 시작
4. **간단한 로그인**: 개발 중에는 하드코딩된 2개 계정으로 자동 로그인
5. **사진 업로드**: 최대 10장, 장당 5MB 제한
6. **AI 리포트**: 등록 시 실시간 생성 (로딩 표시)
7. **가격 필터**: 최소/최대 입력 방식
